# 🎓 Previsão de Desempenho de Alunos

> Prevê desempenho de alunos em provas e no ENEM com ML. Identifica lacunas de aprendizagem por competência e recomenda plano de estudo personalizado para cada estudante.

## ⚙️ Como funciona
1. **Coleta de dados reais** — integração com ENEM/INEP + plataforma de estudo
2. **Processamento com IA** — Machine Learning · Knowledge Tracing · ML
3. **Dashboard & insights** — KPIs, gráficos e recomendações acionáveis
4. **Ação automatizada** — alertas, predições e decisões em tempo real
